Programming Basics

  • Anything that isn't a primitive data type is an object, including functions
  • Avoid "var", use "let" instead. It allow variable to be locally scoped
  • Javascript has "truthy" and "falsy" values. Be careful when doing comparisions. Try to cast to specific types.
  • Use === and !== to check if the value type and the values are the same.

Arrays, Logic and Loops

  • Javascript arrays are ordered list of values
  • Avoid using the delete operator on arrays. It only sets the value to undefined. It doesn't remove the element.
  • Array methods and properties:
    • length - number of elements in an array
    • pop() - remove item from end of an array
    • push() - add item to end of an array
    • shift() - remove item from start of an array
    • unshift() - add item to start of an array
    • concat() - merge arrays
    • join() - merge items in an array into a string
    • slice() - remove a section of an array
    • splice() - add a section to an array
    • reverse() - reverse the order of an array
    • sort() - sort an array, default is alphanumeric
    • indexOf() - get the index of a value in an array
    • includes() - checks to see if a value is in an array
  • Set - holds a unique set of value, similar to a hashset in other languages
  • WeakSet - only holds non-primitive values. Values are removed if garbage collected.
  • Maps - contains key value pairs. Similar to Dictionaries.
  • WeakMaps - only holds non-primitive values. Keys can't be primitives. Values are removed if garbage collected.
  • Do While, While, For are the same as C variants
  • For-Of is like a foreach() for(const car in cars)...
  • use For-Of for looping through Sets and Maps

Functions

  • Function Definitions
    • Function declaration -- function hello(){};
    • Function expressions - Anonymous -- const hello = function(){};
    • Function expressions - Named -- const hello = function hello(){};
  • Avoid using the function constructor Function()
  • DRY - Don't Repeat Yourself - try to write code once in a function. Use the function in multiple places
  • Functions can take more arguments than they have parameters defined for. The extra arguments can be found in the "arguments" object
  • Arrow functions: const square = x => x*x, add = (x,y) => x+y. Similar to Lambda functions.
  • Callbacks are functions passed as arguments to another function
  • The forEach method runs each entry in an array through a function.
  • The map function (not Map data structure) will also iterate over an array updating each item in the array based on a function
  • The reduce method, known as aggregate in other languages, will turn the values in an array into a single value based on another function
  • The filter method will return only values that return true from anther function
