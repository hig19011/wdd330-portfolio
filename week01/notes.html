Local Storage

  • Local Storage records information using Key-Value pairs through page refreshes.
  • The Key must be unique in localstorage for the subdomain.
  • The Value can be repeated.
  • Local Storage can be accessed using the getItem and setItem functions.

UX Design - Mobile First

  • Always design for mobile first.
  • Mobile first requires a simple interface which better focuses the developer on content.
  • Put the most highly used feature in the bottom left corner of a mobile screen and work out from there. This gives the user the easiest access to those features. This includes the navigation menu.
  • Don't required more information than necessary in forms.
  • Use placeholders or embedded labels.
  • Auto advance users through input fields.
  • Use the correct keyboard for the right purpose.
  • Don't use popups or in between page ads.
  • Don't use more than 40% of the screen for an ad.
